Belajar HTML-CSS

HTML digunakan untuk membuat tampilan website. CSS digunakan untuk mengatur tampilan.

Materi HTML

  • "h1" untuk judul
  • "p" untuk paragraf
  • "img" untuk gambar
  • "table" untuk tabel
  • "input" untuk form
  • "li" untuk list / daftar

Materi CSS

  1. Margin, Padding, Position Untuk mengatur jarak antara elemen HTML
  2. "Display" Untuk mengatur elemen HTML agar tampil atau tidak tampil, mendatar (Column), atau menurun (row)
  3. "Style" Untuk mengatur font
